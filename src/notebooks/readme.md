# src/notebooks

Notebooks Jupyter com o pipeline completo CRISP-DM:

- `01_exploracao.ipynb`: EDA — estatisticas descritivas, histogramas, boxplots, scatterplots, correlacao.
- `02_classificacao.ipynb`: treinamento e comparacao de 5+ algoritmos (KNN, SVM, Random Forest, Naive Bayes, Logistic Regression).
- `03_otimizacao.ipynb`: Grid Search e Randomized Search para hiperparametros, reavaliacao dos modelos otimizados.
- `04_interpretacao.ipynb`: interpretacao de resultados, insights e conclusoes.
